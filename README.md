# Updated Quiz App

A modern quiz application built with React, TypeScript, Vite, Tailwind CSS, and the OpenTDB API.

## Features

- Live quiz questions from the OpenTDB API
- Category selection
- Difficulty selection
- Animated screen transitions with Framer Motion
- Progress tracking
- Score system
- Responsive modern UI
- Built with TypeScript

---

# Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- OpenTDB API
